## Switch-Over Sunday Checklist

**Use this checklist the first time you use the platform for Sunday bulletin.**

### Week Before (7 Days Out)

- [ ] **Admin account created** - Verify you can log in
- [ ] **Brand pack configured** - Logo, colors, fonts uploaded
- [ ] **Funds created** - General, Missions, Building (at minimum)
- [ ] **This Sunday's service plan drafted** - Songs, readings, sermon
- [ ] **Announcements reviewed** - Decide what goes in bulletin vs. verbal only
- [ ] **Test bulletin built** - Generate preview PDF
- [ ] **Print test** - Verify kiosk can print the PDF
- [ ] **Backup plan ready** - Keep old process available just in case

### Tuesday (5 Days Before Sunday)

- [ ] **Calendar reviewed** - Events for next 2 weeks entered
- [ ] **Service plan finalized** - All songs have CCLI numbers
- [ ] **Announcements entered** - Max 6 (top 3 get full display)
- [ ] **Images uploaded** - Pastor photo, event graphics (max 4MB each, 16:9 ratio)

### Wednesday (4 Days Before)

- [ ] **Build preview bulletin** - Review for typos, formatting
- [ ] **Share preview link** - Send to pastor for review
- [ ] **Fix any issues** - Edit announcements, adjust service items
- [ ] **Verify CCLI numbers** - All songs must have valid CCLI# before lock

### Thursday (3 Days Before) - **LOCK DEADLINE**

- [ ] **Final review** - Pastor approves bulletin
- [ ] **Lock bulletin** - Click "Lock Bulletin" (default: 2:00 PM)
- [ ] **Download PDF** - Save final bulletin
- [ ] **Send to printer** - If printing commercially
- [ ] **Upload to kiosk** - Or set up auto-print Sunday morning
- [ ] **Export slide loop** - For lobby screen (MP4 or JPGs)
- [ ] **Send email version** - Platform can send to members automatically

### Friday (2 Days Before)

- [ ] **Verify print job** - Confirm commercial printer received file
- [ ] **Test kiosk** - Print one copy to verify it works
- [ ] **ProPresenter setup** - Import slide loop or service plan

### Saturday (1 Day Before)

- [ ] **Final check** - Do you have enough printed copies?
- [ ] **Kiosk ready** - Loaded with paper, toner checked
- [ ] **Slide loop running** - Test lobby screen

### Sunday Morning

- [ ] **Print bulletins** - From kiosk or use pre-printed copies
- [ ] **Verify slide loop** - Running in lobby
- [ ] **ProPresenter loaded** - Service plan ready for worship team

### After Service (Debrief)

- [ ] **What worked well?** - Note for next week
- [ ] **What needs improvement?** - Address before next bulletin
- [ ] **Emergency reopen needed?** - If yes, note the reason for future

## "One Form or It Doesn't Happen" Policy

**Why This Matters:**

Too often, last-minute changes bypass the bulletin process. This creates:
- Confusion (announcement on screen but not in print)
- Wasted time (re-printing bulletins)
- Missed information (people don't know about events)

**The Rule:**

> **If it's not in the bulletin intake form by Thursday 2 PM, it doesn't go in the bulletin.**

**Exceptions:**
- **Emergency only:** Death in congregation, service cancellation due to weather
- **Emergency reopen:** Pastor can reopen locked bulletin, but a timestamp is added to every output showing it was changed after lock

**How to Enforce:**
1. **Communicate deadline** - "Bulletin announcements due Thursday 2 PM"
2. **Post deadline visibly** - On church calendar, in staff meetings
3. **Be firm** - If someone asks Friday, answer is "It will be in next week's bulletin"
4. **Empower admin** - Give bulletin coordinator authority to say no

**Benefits:**
- âœ… Consistent process
- âœ… Reduced stress
- âœ… Higher quality bulletins
- âœ… Fewer last-minute changes

## Troubleshooting Guide

### Issue: Kiosk Won't Print

**Symptoms:** Blank pages, error messages, or no print job starts

**Solutions:**
1. **Check basics:**
   - [ ] Is printer powered on?
   - [ ] Is it connected to network/computer?
   - [ ] Does it have paper?
   - [ ] Does it have toner/ink?

2. **Restart printer:**
   - [ ] Power off, wait 30 seconds, power on
   - [ ] Try printing test page from printer menu

3. **Check file:**
   - [ ] Can you open the PDF on computer?
   - [ ] Try printing from computer instead of kiosk
   - [ ] Re-download PDF from platform

4. **Kiosk specific:**
   - [ ] Restart kiosk computer
   - [ ] Check default printer settings
   - [ ] Verify PDF reader is installed (Adobe Reader recommended)

5. **Workaround:**
   - Email PDF to yourself
   - Print from office computer
   - Use smartphone to print via email/AirPrint

**Still not working?** Call tech support or use backup printed copies.

### Issue: "CCLI Number Required" Error

**Symptoms:** Can't lock bulletin, error message says song missing CCLI

**Solution:**
1. **Identify the song:**
   - Error will show which song is missing CCLI number
   - Example: "Amazing Grace (position 2) requires CCLI number"

2. **Look up CCLI number:**
   - Go to SongSelect.com
   - Search for song title
   - Copy the CCLI number (e.g., 22025)

3. **Add to bulletin:**
   - Edit the service item
   - Paste CCLI number in "CCLI #" field
   - Save changes

4. **Try lock again:**
   - Click "Lock Bulletin"
   - Should succeed now

**Don't have SongSelect account?**
- Most hymns have CCLI numbers (Google "hymn name CCLI number")
- For original songs, leave CCLI blank and note in description
- Consider getting CCLI license (required for churches that project lyrics)

### Issue: "Image Too Large" Error

**Symptoms:** Upload fails with "File must be under 4MB"

**Solutions:**
1. **Check file size:**
   - Right-click image â†’ Properties â†’ Size
   - If over 4MB, needs compression

2. **Compress image:**
   - **Online tool:** TinyPNG.com or CompressJPEG.com
   - **On computer:**
     - Windows: Open in Paint â†’ Resize â†’ Save
     - Mac: Open in Preview â†’ Tools â†’ Adjust Size â†’ Export

3. **Recommended settings:**
   - **Max width:** 1920px
   - **Format:** JPEG (not PNG for photos)
   - **Quality:** 80%
   - **Target size:** Under 2MB

4. **Aspect ratio:**
   - Platform expects 16:9 (landscape)
   - Will crop other ratios automatically
   - Use Canva or PowerPoint to create 1920x1080 canvas

**Still too large?**
- Take screenshot of image at smaller size
- Use Canva to resize before downloading

### Issue: Announcement Text Cut Off

**Symptoms:** Announcement doesn't fit, shows "..." ellipsis

**Cause:** Character limits
- Title: 60 characters max
- Body: 300 characters max

**Solutions:**
1. **Shorten text:**
   - Remove filler words ("very", "really", "just")
   - Use contractions ("don't" instead of "do not")
   - Use abbreviations ("Wed" instead of "Wednesday")

2. **Split announcement:**
   - If too much info, make it two announcements
   - Or link to full details on website

3. **Use full bulletin section:**
   - Top 3 announcements get full display
   - Remaining show as compact list
   - Prioritize most important

**Character counter:**
- Platform shows character count as you type
- Green = OK, Yellow = Warning, Red = Too long

**Example:**

âŒ **Too long (72 chars):**
> "Join us for our annual church-wide potluck dinner this coming Sunday evening"

âœ… **Good (58 chars):**
> "Church potluck this Sunday at 6 PM. Bring a dish to share!"
